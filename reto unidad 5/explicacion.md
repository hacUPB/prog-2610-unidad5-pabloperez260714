## Explicacion 
### introduccion
primero se despliega un menu principal en el cual nos arroja 4 opciones (explorador, txt, csv, salir)
este menu principal lo manejamos con un archivo main.py en el cual se ponen el sistema de menu en el cual el usuario puede escoger las opciones

### opcion 1
en esta opcion vamos a poder ver el explorador, donde podemos encontrar la carpeta data y visualizar el archivo txt y cvs

### opcion 2 (txt)
en esta opcion podemos ver el archivo tipo texto quer esta en la carpeta data, con los requerimiento pedidos, en los cuales se incluyen graficos y los datos escritos de palabras mas usadas, etc.

### opcion 3
en la opcion 3 podemos trabajar con el archivo csv, el cual funciona con listas ordenadas, como una tabla de excvel, esta opcion tambien incluye graficos del mismo

### uso de graficos.py
usamos este para podernos orientar para graficar el archivo txt y csv
